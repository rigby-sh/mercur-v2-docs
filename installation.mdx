---
title: "Installation"
description: "Install Mercur on your local machine"
---

<Info>
  **Prerequisite**: Node.js (version 20 or higher), Git CLI tool, PostgreSQL
</Info>

Follow these steps to install and run Mercur on your operating system:

**Step 1**: Clone the Mercur repository:

```bash
git clone https://github.com/rigbyjs/mercur.git
```

**Step 2**: Navigate to the Mercur directory (where the `package.json` file is
located) and execute the following command:

```bash
yarn install
```

**Step 3**: Start the development server:

```bash
yarn dev
```
